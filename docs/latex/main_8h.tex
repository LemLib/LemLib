{\footnotesize\ttfamily \#define PROS\+\_\+\+USE\+\_\+\+SIMPLE\+\_\+\+NAMES}

If defined, some commonly used enums will have preprocessor macros which give a shorter, more convenient naming pattern. If this isn\textquotesingle{}t desired, simply comment the following line out.

For instance, E\+\_\+\+CONTROLLER\+\_\+\+MASTER has a shorter name\+: CONTROLLER\+\_\+\+MASTER. E\+\_\+\+CONTROLLER\+\_\+\+MASTER is pedantically correct within the PROS styleguide, but not convienent for most student programmers.
