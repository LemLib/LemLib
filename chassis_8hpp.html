double lemlib::defaultDriveCurve ( double  input,
double  scale

Default drive curve. Modifies the input with an exponential curve. If the input is 127, the function will always output 127, no matter the value of scale, likewise for -127. This curve was inspired by team 5225, the Pilons. A Desmos graph of this curve can be found here: https://www.desmos.com/calculator/rcfjjg83zx.

Parameters
inputvalue from -127 to 127
scalehow steep the curve should be.
Returns
The new value to be used.
