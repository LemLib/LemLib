Detailed Description

Feedforward, Acceleration, Proportional, Integral, Derivative PID controller.

The controller does not loop on its own. It must be called in a loop. For example: while(!controller.settled) { controller.update(input, output); }

initialize the FAPID logging system

Enable logging the user can interact with the FAPID through the terminal the user can access gains and other variables with the following format: <name>.<variable> to get the value of the variable <name>.<variable>_<value> to set the value of the variable for example:z pid.kP_0.5 will set the kP value to 0.5 list of variables thats value can be set: kF, kA, kP, kI, kD list of variables that can be accessed: kF, kA, kP, kI, kD, totalError list of functions that can be called: reset()
